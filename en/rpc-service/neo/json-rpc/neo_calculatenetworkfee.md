---
title: neo:calculatenetworkfee \[POST\] {disallowed}
description: Calculates network fee for the specified transaction.
---

### Parameters


`tx` - string

Base64-encoded string of transaction information.

### Request

``` java
curl --location --request POST 'https://neo.getblock.io' 
--header 'x-api-key: YOUR-API-KEY' 
--header 'Content-Type: application/json' 
--data-raw '{"jsonrpc": "2.0",
"method": "calculatenetworkfee",
"params": [null],
"id": "getblock.io"}'
```

###  Response

``` java
{
    "result": "null",
    "id": "getblock.io",
    "status_code": 405,
    "message": "Method not allowed"
}
```
