### Parameters

`height` - unsigned int

The block's height.

`hash` - string

The block's hash.

### Request

``` java
curl --location --request POST 'https://xmr.getblock.io/mainnet/json_rpc' \ 
--header 'x-api-key: YOUR-API-KEY' \ 
--header 'Content-Type: application/json' \ 
--data-raw '{"jsonrpc": "2.0",
"method": "get_block",
"params": {"height": 912345},
"id": "getblock.io"}'
```
