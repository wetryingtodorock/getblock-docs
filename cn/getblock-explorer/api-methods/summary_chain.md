---
title: summary_chain \[GET\]
description: Returns actual blockchain info
---

### Example

```GET /summary/chain```

### Request

```
curl --location --request GET 'https://near.getblock.io/explorer/summary/chain' \
--header 'Accept: application/json' \
--header 'x-api-key: YOUR-API-KEY'
```
